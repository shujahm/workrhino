Installation

Step 1:

Upload main zip into your hosting & then extract that zip file in same directory.

Step 2:

Then create a database and import SQL file into it. SQL file available in "database" folder

Step 3:

Open .env its available in local folder & then change hostname, database name & password.

And Open database.php its available in local/config folder & then change hostname, database name & password.

Step 4:

Go to Admin panel http://yoursite.com/admin

username: admin
password: admin

Step 5:

Then goto here: http://yoursite.com/admin/settings

Change site name,logo & favicon informations change here. That's all

Step 6:

Server Requirements

PHP version : use 5.6 and above
